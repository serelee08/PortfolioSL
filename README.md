# AI Backend Portfolio

AI 기반 서비스의 요청 처리 흐름과 파이프라인 설계에 집중한  
백엔드 엔지니어 이지애의 포트폴리오입니다.

---

## Overview

본 레포지토리는 AI 기능이 실제 서비스 흐름 속에서 안정적으로 작동하도록  
백엔드 구조를 설계·구현한 프로젝트 경험을 정리한 포트폴리오입니다.

---

## Core Focus

- 요청 → API → AI 모델 → DB → 응답으로 이어지는 백엔드 처리 흐름 설계
- 구조가 명확한 파이프라인을 통한 서비스 안정성 및 확장성 확보
- 팀 협업을 고려한 API 문서화와 인터페이스 정리

---

## Tech Stack

- **Language**: Python, Java(Spring), TypeScript  
- **Backend**: Django, Spring Boot  
- **AI**: Gemini API, LSTM  
- **Database**: MySQL, MariaDB, Oracle  
- **Infra**: AWS EC2 (Linux)  
- **Tools**: Git, GitHub, Postman

---

## Projects

### EVLink – AI 기반 전기차 충전기 공유 플랫폼

- **기간**: 2025.08 – 2025.09  
- **형태**: 팀 프로젝트  
- **역할**: 백엔드 / AI 기능 구현

**What I Did**
- Django 기반 Gemini AI 대화형 서비스 REST API 구현
- LSTM 감정분석 모델 설계 및 백엔드 서비스 연동
- 사용자 요청 처리 로직 및 REST API 설계·개발
- OAuth2 기반 인증 구조 설계
- 데이터 모델링 및 DB Schema 설계
- API 문서화로 팀 협업 및 유지보수 효율 개선
- AWS EC2(Linux) 환경에서 서비스 배포·운영

---

### Walley – 통계 기반 자산 관리 플랫폼

- **기간**: 2025.05 – 2025.07  
- **형태**: 팀 프로젝트  
- **역할**: 백엔드 / 데이터 처리 로직 구현

**What I Did**
- Spring Boot 기반 RESTful API 설계 및 수입·지출 처리 로직 구현
- MyBatis 기반 Oracle DB 연동 및 데이터 모델링
- 지출 내역 영수증 이미지 업로드 및 관리 기능 구현
- 카테고리·기간·시간 기준 수입·지출 조회 API 개발
- React + TypeScript 프론트엔드와 API 연동
- Oracle Database Schema 설계 및 쿼리 작성
- Postman을 활용한 API 테스트 및 데이터 처리 검증



## Architecture

Client → API Server → AI Model → Database → Response  
요청 처리 흐름 전반을 고려한 백엔드 아키텍처로 설계했습니다.
